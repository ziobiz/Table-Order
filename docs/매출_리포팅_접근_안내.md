# 매출 리포팅 접근 안내

## 1. 접근 조건

- **본사(SUPERADMIN) 로그인** 필요
- 비로그인 또는 다른 역할로 접근 시 → 로그인 페이지로 리다이렉트

## 2. 접근 경로 (매출 리포팅 링크 위치)

### 본사 대시보드 (admin_dashboard.php)

| 레이아웃 | 위치 |
|----------|------|
| **타블렛 모드** (카드 그리드) | "매출 리포팅" 카드 — Settlement 카드 다음, Trx Ledger 카드 앞 (7번째 카드) |
| **PC 업무 모드** (사이드바) | 왼쪽 사이드바 — "매출 리포팅" 메뉴 (Settlement 다음, Trx Ledger 앞) |

### 직접 URL

- [매출 리포팅](https://wirexcard.net/admin_revenue.php) — **본사 로그인 후** 접근

## 3. 본사 로그인 방법

1. [로그인](https://wirexcard.net/login.php) 접속
2. **HQ** 탭 선택
3. 본사 계정(admins 테이블)으로 로그인
4. 로그인 후 [본사 대시보드](https://wirexcard.net/admin_dashboard.php)로 이동
5. "매출 리포팅" 카드 또는 사이드바 메뉴 클릭

## 4. 페이지가 보이지 않을 때

- **본사(HQ) 탭**으로 로그인했는지 확인
- `admin_revenue.php`, `admin_dashboard.php`, `admin_header.php` 파일이 서버에 업로드되었는지 확인
- 브라우저에서 [매출 리포팅](https://wirexcard.net/admin_revenue.php) 직접 접속 시 "최고관리자만 접근 가능합니다" 알림이 뜨면 → HQ로 로그인 후 다시 시도
